# nixe

Another high-level browser automation library.

Heavily inspired by [nightmare][1]


## Compared To Nightmare

- [x] ES2015+ Build System
- [x] Simpler Codebase
- [ ] Seamless Access To Electron Objects


## Dev & Test

```plain
$ npm install -g webpack mocha
$ cd nixe
$ npm install
$ npm run dev
$ npm test
```
